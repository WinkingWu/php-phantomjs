PHP PhantomJS
=============

PHP PhantomJS是一个灵活的PHP库，可以通过PhantomJS无头浏览器加载页面并返回页面响应。它对于测试需要javascript支持并且还支持屏幕捕获的网站非常方便。

[文档](http://jonnnnyw.github.io/php-phantomjs/)
